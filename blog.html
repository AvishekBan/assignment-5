Important FAQ


What is Document Object Model(DOM)?

The document object model is represents the structure of a web page and with the help of javascript using DOM we can access,change,manipulate,delete and add different parts of webpage in dynamic way.

How do you select an element from the DOM?

I can select an element from DOM in multiple way.such as: getElementById, getElementsByClassName, getElementsByTagName, querySelector, querySelectorAll

What is the event delegation in the context of the DOM, and why is it useful?

Event-bubbling is the foundation of event delegation. With the help of this event-handling method, you can manage events that were initially received at a different level of the DOM tree.That's why its necessary.

How do you manipulate an element's attributes and styles using the DOM?

The values of an element's style attribute are returned via the style property. A CSSStyleDeclaration object is the result of the style property.
